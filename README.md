# telegram-gpt-go

---

## go.mod 사용법
```bash
# go.mod 파일 생성
go mod init github.com/motolies/telegram-gpt-go

# go 의존성 추가
# 처음 한 번만 실행하면 됨
go get github.com/go-telegram-bot-api/telegram-bot-api
```
